Detailed Description

Si4703 FM radio module library implementation based on Tomas Fryza's TWI library.

Si4703 FM radio module library based on Tomas Fryza's TWI library.

#include <si4703.h>
si4703.h
#include <si4703.h>

Library for interacting with the Si4703 FM radio module. This library was developed for the purposes of a DE2 class project and does not offer full functionality.

inspired by the SparkFun Si4703 Arduino library and the bare metal AVR_SI4703 library from github user eziya

Developed for ATmega328p/Arduino Uno R3
